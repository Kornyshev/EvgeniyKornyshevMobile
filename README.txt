    Template for file src/main/resources/credentials/cloud.properties
    url = https://EPM-TSTF:token@mobilecloud.epam.com/wd/hub
    token = YOUR_TOKEN
    ios.udid = IOS_SERIAL_NUMBER
    android.udid = ANDROID_SERIAL_NUMBER

    Теперь основная часть README:
    1. Чтобы не держать токен в Git я просто добавил файл с настройками в gitignore
        (конечно, можно было ещё разделить его на два файла - один с токеном, другой с серийниками
        чтобы при клонировании репозитория к себе добавлять только токен
        но это уже, наверное, слишком)
    2. То есть чтобы добавить свой токен и серийники девайсов нужно будет создать файл
        src/main/resources/credentials/cloud.properties по шаблону выше
    3. Соответственно запуск тестов реализован через профили Maven и почти все параметры передаются
        через файлы XML

    4. Здесь реализована установка приложения по API, так что будет чуть проще обращаться с девайсами в облаке
    5. Но так и не получилось нормально заставить работать Веб тест на iOS девайсах
